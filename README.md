# scroll-num

> '数字翻滚效果'

## npm 引入

``` bash
npm install scroll-num-vue

yarn add scroll-num-vue

## npm使用
`
import scroll from 'scroll-num-vue';
app.use(scroll);
`

### props

```
num  需要显示的数字
dot  是否显示逗号
